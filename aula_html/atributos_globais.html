Atributos Globais

Diferença entre ID e Class

ID: Como se fosse um RG, não pode-se ter repetido. Cada tag tem a sua

Class: É uma classificação, pode repetir e colocar espaço, mas a cada espaço é como se fosse uma segunda classificacao

Exemplos

id="texto1" class="produto"

id="texto2" class="produto destaque"

id="texto3" class="produto"

id="texto4" class="produto"

id="texto5" class="produto destaque"

Data-*

O data-* é um identificado só que para numericos

Exemplo: data-preco="100"

Exemplo: data-idade="18"

Exemplo: data-codigo="12345"

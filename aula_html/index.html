Caracteres Especiais

Para colocar o < no HTML, sem que o HTML identifique como uma abertura de tag, usamos o comando:

Para direita: & lt ; (tirar os espaços)

para esquerda: & gt ; (tirar os espaços)

Exemplos:

<lt>Conteúdo<gt>

<div>Teste</div>

<p>Lorem</p>

Atributo Booleano

Para um atributo ser booleano, deve-se colocar hidden

Exemplo:

<div hidden>Conteúdo oculto</div>

O conteúdo acima não será exibido na página
